# Web Scraper for News Headlines

## Description

This project is a Python-based News Headlines Web Scraper developed as part of my Python Developer Internship Task 3.

The application fetches news headlines from a public news website, extracts the headlines using BeautifulSoup, and stores them in a text file.

---

## Objective

To automate the collection of news headlines from a public website using Python web scraping.

---

## Features

* Fetches HTML content from a news website
* Extracts news headlines automatically
* Saves headlines into a text file
* Uses BeautifulSoup for HTML parsing
* Handles request errors using try-except blocks

## Installation

Install the required libraries:

pip install beautifulsoup4

---

## How to Run

Open the project folder in VS Code.

Open the terminal and run:

python news_scraper.py

---

## Output

After running the program:

* News headlines are fetched from the website.
* Headlines are saved into:
