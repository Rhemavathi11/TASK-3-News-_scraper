# Web Scraper for News Headlines

## Description
This is a Python web scraper created as part of my Python Developer Internship Task 3.

The program fetches news headlines from a public news website and saves them into a text file.

## Features
- Fetches HTML content from a news website
- Extracts headlines using BeautifulSoup
- Saves headlines into `headlines.txt`
- Handles request errors using try-except
